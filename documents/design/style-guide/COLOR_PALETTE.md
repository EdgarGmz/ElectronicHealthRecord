## 🎨 Colores con Transparencia

Útiles para overlays, backgrounds, y efectos visuales.

### Primary con Alpha

```css
/* Backgrounds sutiles */
rgba(59, 130, 246, 0.05)  /* 5% */
rgba(59, 130, 246, 0.10)  /* 10% */
rgba(59, 130, 246, 0.20)  /* 20% */

/* Overlays */
rgba(59, 130, 246, 0.40)  /* 40% */
rgba(59, 130, 246, 0.60)  /* 60% */

/* Sólido */
rgba(59, 130, 246, 0.90)  /* 90% */
rgba(59, 130, 246, 1.00)  /* 100% */
```

### Black/White con Alpha

```css
/* Sombras y overlays oscuros */
rgba(0, 0, 0, 0.05)   /* Sombra sutil */
rgba(0, 0, 0, 0.10)   /* Sombra ligera */
```

## ✅ Checklist de Validación

Al implementar colores, verificar:

- [ ] Contraste mínimo 4.5:1 para texto normal
- [ ] Contraste mínimo 3:1 para texto grande (≥18px)
- [ ] Contraste mínimo 3:1 para elementos UI
- [ ] Color no es el único indicador (usar íconos/texto también)
- [ ] Colores probados en modo claro y oscuro (si aplica)
- [ ] Accesible para personas con daltonismo
- [ ] Imprimible en escala de grises si es necesario
