# Matriz RACI - Electronic Health Record (EHR)

**Proyecto:** Sistema de Registro de Salud Electrónico (EHR)  
**Documento solicitado:** Matriz RACI *Nombre del proyecto*.xlsx  
**Nombre del archivo a entregar:** `Matriz RACI Electronic Health Record.xlsx`

---

## Archivos incluidos

| Archivo | Uso |
|---------|-----|
| **Matriz RACI Electronic Health Record.xlsx** | Archivo listo para entregar. Todos los integrantes del equipo deben subir este archivo. |
| **Matriz-RACI-Electronic-Health-Record.csv** | Versión en CSV (respaldo o edición en otra herramienta). |
| **Matriz-RACI-Electronic-Health-Record.xml** | Versión en Excel 2003 XML (alternativa para abrir en Excel y guardar como .xlsx). |

---

## Cómo obtener el archivo .xlsx para entregar

1. Abrir **Microsoft Excel**.
2. **Archivo → Abrir** y seleccionar:
   - `Matriz-RACI-Electronic-Health-Record.xml`, **o**
   - `Matriz-RACI-Electronic-Health-Record.csv` (si usa CSV, elegir delimitador “Coma” y codificación UTF-8).
3. **Archivo → Guardar como**.
4. Nombre del archivo: **Matriz RACI Electronic Health Record.xlsx**
5. Tipo: **Libro de Excel (*.xlsx)**.
6. Guardar en la carpeta que vaya a subir (o en la que indique el profesor).
