# Documentación del proyecto EHR

Documentos del proyecto **Electronic Health Record (EHR) System** organizados por tipo.

---

## Estructura de carpetas

| Carpeta | Contenido |
|---------|-----------|
| **[gestion-proyecto/](./gestion-proyecto/)** | Acta de constitución, Plan de Trabajo (MS Project), Matriz RACI. |
| **[requisitos/](./requisitos/)** | Requisitos funcionales, no funcionales, seguridad y reglas de negocio. |
| **[riesgos/](./riesgos/)** | Análisis de riesgos y amenazas, FMEA/AMEF y script de generación. |
| **[diseno-tecnico/](./diseno-tecnico/)** | Diagramas de flujo, flujo de navegación y esquema de Web Services y autenticación. |

---

## Contenido por carpeta

### gestion-proyecto/
- Acta de constitución del proyecto
- Plan de Trabajo (documento y CSV para MS Project)
- Matriz RACI (CSV, XML y Excel)

### requisitos/
- Requisitos funcionales
- Requisitos no funcionales
- Requisitos de seguridad
- Reglas de negocio

### riesgos/
- Análisis de riesgos y amenazas
- FMEA – Electronic Health Record.xlsx
- Script para regenerar el FMEA y README

### diseno-tecnico/
- Diagrama de flujo de atención
- Flujo de navegación completo
- Esquema de Web Services y autenticación
